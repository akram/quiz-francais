Informacion de tipo de usuario
El tipo de usuario Admin es encargado de todo el sistema y tiene acceso completo. Sub-Admin tiene acceso a crear encuestas de un departamento en especifico y tiene la capacidad de realizar encuestas. El tipo de usuario regular solamente tiene acceso a realizar encuestas que se le han asignado.
